🧪 H-120 Reagent & Consumables Guide

Complete specifications, roles, and troubleshooting for all reagents

🔬 Reagent System Overview

The H-120 uses a gradient elution HPLC system with 3 main consumables that work together to separate and quantify hemoglobin variants. Understanding each reagent's role is crucial for optimal performance and troubleshooting.

Quick Reference: Reagent Consumption

Traditional HPLC 6 reagent cycles per test
H-120 (Patented) 0.3 reagent cycles per test
Savings 95% less reagent waste! 💰
🧬

Hemolysis Solution (H-12H)

Lyses red blood cells to release hemoglobin for analysis

🎯 Primary Role: Breaks open RBCs → releases hemoglobin → prepares sample for HPLC injection

📋 Specifications

Parameter Value
Package 5.6L bottle
Storage 2-8°C (refrigerated)
Shelf Life 12 months unopened
pH 10.4-10.6
Osmolality 270-290 mOsm/kg

⚙️ How It Works

Whole Blood (10µL)
+
Hemolysis Solution
→
RBCs Rupture
→
Hb Released

Key Components:

  • Non-ionic detergent (breaks cell membrane)
  • Carbonate buffer (pH ~10.5)
  • Osmotic agent (prevents Hb denaturation)
  • Preservative (prevents bacterial growth)
⚠️ Critical Factor: pH must stay between 10.4-10.6. If pH drifts, hemoglobin charge changes, affecting separation! Check pH weekly with calibrated meter.

🔧 Troubleshooting

❌ Hemolyzed sample looks brown/dark
🔍 Cause: Oxidized hemoglobin (old solution)
✅ Fix: Replace solution bottle, check refrigeration
❌ Inconsistent results between tubes
🔍 Cause: Uneven hemolysis (wrong volume ratio)
✅ Fix: Verify 1:10-1:20 sample to hemolysate ratio
🌊

Eluent A (H-12A) - Primary Buffer

Low-salt buffer that elutes weakly-bound hemoglobin first

🎯 Primary Role: Creates low-ionic environment → HbA1c elutes FIRST (~1.5 min) → begins gradient transition

📋 Specifications

Parameter Value
Package 1.5L bottle
Storage Room temp (15-25°C)
Shelf Life 12 months unopened
pH 8.9-9.1 (CRITICAL!)
Ionic Strength Low (~0.05 M)

🔬 Gradient Elution Concept

How Eluent A enables peak separation:

Eluent A (LOW salt): Weak Hb charge → Binds to column → HbA1c elutes first

Gradient transition: Salt ↑ gradually → Variants elute in sequence

Eluent B (HIGH salt): Strong Hb charge → Cannot bind → HbA0, HbA2 elute last

⚠️ pH SENSITIVITY:
  • pH 8.5: HbA1c elutes too fast ❌
  • pH 8.9-9.1: OPTIMAL separation ✅
  • pH 9.3: HbA1c elutes too slowly ❌

🔧 Troubleshooting

❌ HbA1c eluting at wrong time
🔍 Cause: Eluent A pH drifted (< 8.8 or > 9.2)
✅ Fix: Replace Eluent A; verify pH weekly
🧂

Eluent B (H-12B) - High-Salt Buffer

High-salt buffer that repels hemoglobin from column (complete elution)

🎯 Primary Role: High salt concentration → Elutes all remaining Hb → Column cleanup → Ready for next sample

📋 Specifications

Parameter Value
Package 250mL bottle (small!)
Storage Room temp, light-protected
Salt Concentration HIGH (~0.3-0.4 M)
Consumption Much lower than Eluent A

💡 Why Gradient Elution Matters

H-120's Competitive Advantage:

✅ Gradient Elution (H-120)

  • Continuously changes salt concentration
  • Better peak resolution & separation
  • Detects rare hemoglobin variants
  • Better HbA1c accuracy (±2%)

❌ Fixed-Concentration (Competitors)

  • Limited separation resolution
  • Misses rare variants
  • Higher false-positive rates

Analytical Column (Ion-Exchange Resin)

The "heart" of H-120 - separates hemoglobin by electrical charge

🎯 Primary Role: Separates hemoglobin variants by charge → Different retention times → Identified by peaks

📋 Specifications

Parameter Value
Particle Size ~3 µm (industry-leading!)
Type Anion-exchange (strong)
Lifespan 3,000-10,000 tests
Max Pressure 25-30 MPa
Column Length 50-75 mm

🧪 How Resin Separation Works

Column Surface: Negatively charged (-O-SO₃⁻)

HbA1c (charge +1) → Binds weakly → Elutes FIRST

HbS/C (charge +2) → Binds moderately → Elutes MIDDLE

HbA0 (charge +3) → Binds strongly → Elutes LAST

🔧 Column Maintenance is Critical:
  • Daily: System purge (2-3 cycles with Eluent A)
  • Weekly: Inject QC sample, check baseline
  • Monthly: Verify column efficiency (Resolution >1.5)
  • Replace if: CV% > 2% or pressure unstable

🔧 Troubleshooting

❌ Peaks broadened or split
🔍 Cause: Air bubbles in column or contamination
✅ Fix: Run 20-min purge; if not resolved, replace column
❌ CV% gradually increasing
🔍 Cause: Column packing degradation
✅ Fix: Replace column (normal wear after 10K tests)
🔍

In-Line Sample Filter

Removes particulates before column injection

🎯 Primary Role: Filters out RBC remnants, clots, fibrin → Protects column → Maintains baseline stability

📋 Specifications

Parameter Value
Pore Size 10 µm
Material PTFE (Teflon)
Lifespan 3,000 tests
Flow Rate ≥1 mL/min @ 15 MPa

⚠️ Filter Saturation Signs

🔴 Replace filter immediately if:

  • System pressure ↑ by >2 MPa
  • Chromatogram baseline becomes noisy
  • Peak broadening across all samples
